# sigma
SIGMA é uma plataforma de monitoramento de condição industrial baseada em ESP32-S3 para aquisição e análise de dados de temperatura, vibração e operação de ativos industriais.  O projeto aplica conceitos de IIoT, manutenção preditiva e monitoramento em tempo real para auxiliar na identificação precoce de falhas e na tomada de decisão operacional.
